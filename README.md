# Pakistan CPI Analysis 

## Overview
This repository contains the data, statistical analysis, and final deliverables for the **Pakistan CPI Analysis** project. The project investigates the Consumer Price Index (CPI) and related economic indicators for Pakistan, utilizing data from major global financial institutions.

---

## Repository Structure

The project directory is organized as follows:

*   **`data/`**: This directory contains the raw datasets used for the analysis.
    *   `master_IMF.csv`: The master dataset containing economic indicators sourced from the International Monetary Fund (IMF).
    *   `master_WB.csv`: The master dataset containing economic indicators sourced from the World Bank (WB).
*   **`STAT222_Pakistan_CPI_Analysis.ipynb`**: The primary Jupyter Notebook containing the Python code, data cleaning processes, statistical analysis, and generated visualizations.
*   **`STAT222_Pakistan_CPI_Report_Final.docx`**: The comprehensive final written report detailing the methodology, statistical findings, and economic conclusions of the project.
*   **`STAT222_Pakistan_CPI_Presentation.pptx`**: The slide deck used for the final project presentation, summarizing the key insights and data visualizations.

---

## Data Sources
The analysis heavily relies on two primary data files:
1.  **International Monetary Fund (IMF):** (`master_IMF.csv`).
2.  **World Bank:** (`master_WB.csv`).

*Note: Ensure both CSV files remain in the `data/` folder for the relative paths in the Jupyter Notebook to function correctly.*

## Getting Started

### Prerequisites
To run the analysis notebook, you will need Python installed along with the following standard data science libraries:
*   `pandas`
*   `numpy`
*   `matplotlib`
*   `seaborn`
*   `statsmodels` (or other statistical packages used in your specific analysis)
*   `jupyter`

### Installation & Execution
1.  Clone this repository to your local machine:
```bash
    git clone [https://github.com/MuhammadHassan-10/PAKISTAN_CPI_ANALYSIS.git](https://github.com/MuhammadHassan-10/PAKISTAN_CPI_ANALYSIS.git)
    ```
2.  Navigate to the project directory:
```bash
    cd PAKISTAN_CPI_ANALYSIS
    ```
3.  Launch Jupyter Notebook or open the file in VS Code:
```bash
    jupyter notebook STAT222_Pakistan_CPI_Analysis.ipynb
    ```
4.  Run the cells sequentially to reproduce the analysis.

```
